vim的使用

2018-06-10

记录一下自己常常使用的VIM命令，每次都搜索也不是个事儿。

容易忘记的快捷键

  • 显示文件行号：:set nu。

跳转(命令模式)

  • 跳到行首：^。跳到行尾：$.
  • 跳到下一个单词：w,W,e,E。跳到上一个单词：b, B。
  • 跳到上一行首：-。跳到下一行首: +;
  • 跳到下一段： { 。跳到上一段 }。
  • 翻页： ctrl + f 和 ctrl + b;
  • 跳转到光标所在单词下一次出现的地方：n,上一次出现的地方shift + n。
  • 定位至被标记出单词下一次出现的地方:*,上一次出现的地方#。

多文件查看

  • 一次打开多个文件进行查看：vi file1 file2 file2
  • 单个窗口，查看不同文件： b + filename。
  • 单个窗口，查看上一个文件： bp，查看下一个文件 bn。

多窗口查看

  • 多个窗口查看文件：
    水平切分窗口：:split，垂直切分窗口：:vsplit。
  • 多个窗口之间跳转 ctrl + w + h/k，或者ctrl + w + w（w按两次）

补全功能

描述 快捷键
整行补全 CTRL-X CTRL-L
根据当前文件里关键字补全 CTRL-X CTRL-N
根据字典补全 CTRL-X CTRL-K
根据同义词字典补全 CTRL-X CTRL-T
根据头文件内关键字补全 CTRL-X CTRL-I
根据标签补全 CTRL-X CTRL-]
补全文件名 CTRL-X CTRL-F
补全宏定义 CTRL-X CTRL-D
补全vim命令 CTRL-X CTRL-V
用户自定义补全方式 CTRL-X CTRL-U
拼写建议 CTRL-X CTRL-S

查看函数定义

查看函数的定义：命令模式下，gd。
查看完函数定义想返回到原来的地方：命令模式下，ctrl + o。

使用ctags查看程序

上面使用gd只能用来查看函数的定义，但是要查看变量的定义就跳转不了。而Ctags就提供变量，函数，类，结构体，对象，接口等等的快速定位功能。如果用过Visual Studio，你肯定知道F12和ctrl + - 。他们的功能和Ctags提供的功能一样，都是快速定位和快速返回。
