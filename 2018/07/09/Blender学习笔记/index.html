Blender学习笔记

2018-07-09

Blender 学习的笔记

Part 1: User Interface

80/20 Rules

  1. modeling
  2. matrial
  3. lighting

界面

  1. File-New-Reload Start-Up file可以显示最初的界面
  2. Number Pad (小键盘)： 左视图，俯视图等
  3. Mouse+ middle(旋转)， Shift+Mouse(middle) 平移
  4. Mouse(right) :选择
  5. 选择物体+Numer Pad(.) 可以以某个物体为中心旋转

物体

创建物体： shift+a (Mesh,Curve,Surface,Metaball…)
搜索： space + keyworrds
Animation: Alt+A

创建一个燃烧的猴头

  1. Shift+A 选择Mesh-Monkey
  2. Space 搜索 Smoke选择 Quick Smoke
  3. Alt + A 开始动画
  4. 可以在Property面板的Physics控制面板里选择烟的分辨率
  5. 鼠标右键选择猴头，Flow里面可以选择Flow Type: Fire/Fire+Smoke/Smoke等效果。

Part 2: Moving, Rotating, Scaling

Scaling

  1. 按住S键， Ctrl（快） 和 Shift（慢） 可以控制缩放的速度

Moving

  1. 按G键，物体就可以跟着鼠标移动
  2. 按G键，然后再按x/y/z，物体就在x/y/z轴上移动
  3. 按G键，按住鼠标Middle键，物体就可以在x/y/z上移动
  4. 按住鼠标左键可以直接移动物体

Rotating

  1. 按R键，鼠标控制旋转
  2. 按R键，按x/y/z，物体就绕着x/y/z轴旋转

如何将变换设置成Identity?

  • Ctrl + A 然后apply scale/rotation/transltion

一个运动的圆圆圈

  1. Shift + A 选择 mesh + torus
  2. Add Modifier (subdivision + wave)

subdivision modifier 可以增加物体的光滑程度，以后也便于修改
wave 可以让物体运动起来（Alt+A开启动画）

Part 3: Edit Mode

按Tab键可以进入Edit Mode

vertex 模式

  1. 鼠标右键选择vertex
  2. Shift + 鼠标右键可以多选
  3. Alt + 鼠标右键可以选 一圈(row)Edge (也叫Edge Loop)
  4. 选择之后按住G可以移动某个primatives
  5. 加入一些Bumps： 按O键可以进入Propotional Editing, 右键可以移动选择到vertex/edge/face周围的部分。
  6. 第五步的影响范围可以 Mouse Middle来放缩大小。

选择

  • 按住b,然后鼠标左键选择

  • 按住b,然后鼠标中间反选择

  • 按住a,可以选择物体所有的primatives

  • 按住c,可以选择圆圈里的primatives，圆圈的大小鼠标中间放缩可以调节。按住鼠标中间移动可以反选择。

  • 按住z,可以进入wireframe格式

创建甜甜圈

  1. 添加Torus
  2. 选择上面一般网格（alt+shift+鼠标左键）或者（b+前视图选择上半部分）
  3. Shift + D 可以复制刚才那上半部分。 鼠标左键移动，亦可移动上半部分。按Esc可以让上半部分恢复到原来的位置
  4. 如何区分复制部分和原来的部分
    答：选择未复制/复制 部分的一点，按L。可以显示与改点Link的物体。 Ctrl + L : select linked all

最后的例子：无限的torus

  1. 创建一个torus
  2. Add modifiers- Array 数量选择40，取消Relative offset,选择Object Offset 中的Empty
  3. 创建Empty里的Plain Axis
  4. 对torus进行放缩和旋转就可以看到很奇妙的现象了。

Part 4: Material Nodes

Color, Shading like real world
Material Node Editor
上一节我们能够做出一个甜甜圈了，这时候在property面板有个Reander按钮，如果我们点击Reander按钮，就可以看到渲染之后的场景。

在Blender最上面的Panel我们可以选择Reander引擎。里面的选择有：Blender Render, Cycles Render, Blender Game

在最下方的Panel可以选择Slot,显示不同Render引擎渲染出的效果。按J键，可以在slot之间进行变换。

按Esc,可以从渲染面板中退出来。

隐藏

  • Object Mode, 鼠标有右键选择物体，然后按H键。
  • 按Alt+H, 显示出来所有隐藏的物体。

Material

在Property面板里面有Material图标，点击之后可以在里面添加新的material.(需要注意的是：选择Cycles Render和Blender Render出现的选项是不同的。)

如果添加了Materials之后，每次都Render出来看效果调试速度太慢了。我们可以在Object Model左侧选择Rendered选项，每次只要Materials改变了设置，效果直接就出来了。可以按shift+z从其他模式切换到Rendered模式。

如果物体的表面渲染出来不平滑，可以选择smooth。

Material Node Editor

之前设置Material要么简单的设置Diffuse BSDF,要么Gloss BSDF。一般是把他们结合在一起，是的Diffuse, Reflection等等效果叠加在一下来表示物体表面材质属性。

如何进入Material Node Editor

  • 在左下角有个Cube图标，单击之后进入Node Editor

  • 每个Node都有输入输出，颜色相同的可以相连

  • Shift + A 可以添加shader,texture, shader等等

Part 9: Rendering and Compositing

平常我的贴图步骤

  1. 导入文件
  2. 在edit mode下，按c来快速选择需要切割的地方
  3. 按 Ctrl+E 选择 make seam
  4. 按 a两次，全选整个网格，按u(uv mapping)选择unwrap。
  5. 打开UV/Image Editor
  6. 渲染引擎选择Cycles Render
  7. 新建Material，Color选项选择Image Texture，导入纹理图片
  8. 在UV/Image Editor中选择纹理图片，并且里面有切割好的网格
  9. 选择某一部分的网格中一点，按住l可以将和网格连在一起的全选，然后可以进行旋转平移放缩的操作。
  10. 在3D view可以按shift + z来显示渲染出来的效果
